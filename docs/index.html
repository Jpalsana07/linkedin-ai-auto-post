Open source Bun + TypeScript GitHub Actions

LinkedIn AI Agent

A daily LinkedIn post generator that runs entirely on GitHub Actions — no servers, no SaaS bridges, no manual posting.

What it does

Once per day, the agent picks a topic from a weighted pillar rotation, applies a content angle (postmortem, contrarian take, tool comparison, etc.), generates a post via your LLM of choice, validates it against strict rules, and publishes to your LinkedIn account through LinkedIn’s official Posts API.

Architecture

Runtime
Bun 1.2+ (native TypeScript, native .env)
LLM
OpenAI / Anthropic / Gemini behind a single provider interface
Publishing
LinkedIn OAuth 2.0 + Posts API v202402
Scheduling
GitHub Actions cron (free)
Dedup
Jaccard similarity on word + bigram sets
Validation
Word count, hook length, banned phrases, required anchors, semantic similarity

Topic strategy

Three pillars rotated by deficit-based selection (whichever pillar is most under-represented gets picked next):

  • AI Engineering — 45%
  • Architecture and Stack — 35%
  • Founder and Career — 20%

Within each pillar, topics rotate LRU with a 7-day cooldown. Each post is also assigned an angle (postmortem, contrarian-take, tool-comparison, etc.) for structural variety.
